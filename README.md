# Validatify

## Project Overview

Validatify is a simple project that focuses on solidifying skills on form validation.

## Project Architecture
MVC Architecture

### Folder Structure
```
├── index.html
├── src
│   ├── controller
│   │   └── main.js
│   ├── model
│   │   ├── submit-btn.js
│   │   ├── validate-email.js
│   │   └── validate-password.js
│   └── view
│       └── style.css
```


## How to run validatify

1. Clone repo
``` 
git clone URL
```
2. install dependencies
```
npm install
```
3. Run the command
``` 
npm run dev
```
4. Copy local host and send to browser. <br> 
Example: http://localhost:5173/
